Music Player

About the Project

This music app is an interactive cross-platform application developed with Cordova. Users can upload their favourite songs and album art to add to their list. Users can also add a song to their "Top Picks" by clicking on the heart in the top right corner of the song page. The app contains other features such as speaker icons, a footer with details and controls, and an interactive slider on the song page.

Technologies Used

Built with Cordova and designed for Android, this app focuses on the cordova media plugin and handles successes and failures according to it's documentation. It's a Single Page Application (SPA) which means its load time is low, and it responds to user input very quickly. Device Local Storage was cleverly implemented to store and retrieve the pause location of the last song played.

Implementation

Cordova is a useful tool as it allows the developer to build an app using Javascript, HTML5, and CSS then build platform specific web assets for either Android or IOS. Once you run the build command, you simply need to run it on the device you choose and it will download. As this app is for personal use I have not implemented it on any publicly accessible platforms.
