Goobie Games

About the Project

Goobie Games is an interactive cross-platform application developed with React Native. It is an offline, user-focused game list app that has players choose a Name and Avatar, then randomly selects a player to choose a game for the party to play. Once the game is finished, the players will shuffle again and pick more games!

Technologies Used

Built with React Native, Goobie Games utilizes many of the most prominent technologies available in the framework including (but not limited to) React Native core components (View, Text, Flatlist, Keyboard Avoiding View, Pressable, etc) Context, Hooks, Stylesheets, Animation API, Sound API, Alert API, Camera API, GestureHandler API, as well as other Expo technologies.

Implementation

This app was initally published with Expo as part of a school project, but was very quickly after approved on the Google Play Store. A benefit of using Expo for the project is that it makes building APKs quick and easy, allowing for smooth updates on Google Console.
