Github User Search

About the Project

This Github User Search App was built using Swift for IOS. It connect to the gitHub REST API, includes 4 internal and 1 external page, and boasts a number of features including List Views, Page Navigation, Stepper Inputs, as well as image and network libraries. It was also designed to follow the IOS best design practices, and function both in landscape and portrait mode.

Technologies Used

This app includes methods and technologies such as the gitHub search API, JSON fetching, Model Classes, stored user settings (shared Preferences), Localizable Strings, the KingFisher image library, and the AlamoFire Network Library.

Implementation

This App was built as a part of a school project and as such has not been publicly implemented. However, it has been compiled and downloaded on a mobile device for testing and grading purposes.
