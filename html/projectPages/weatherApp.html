The Weather App

About the Project

This weather app is a simple web app using vanilla Javascript, HTML5, and CSS. The use of multiple RESTful APIs allows the user to see the weather based on their current location, or search for weather anywhere in the world. The app also displays general current information, as well as hourly and weekly weather forecasts.

Technologies Used

This web app was built using only basic Javascript, HTML5 and CSS. Its functionality depends on Google's Geolocation API for geolocation and reverse geolocation, as well as openWeatherMap's API for forecasts and weather details. It was also designed to be responsive for nay screen size. Each type of weather also has its own dynamic background.

Implementation

This web app is currently being hosted on Github Pages as a public Repo. It is publicly available, so anyone can visit the site or fork the repo.
