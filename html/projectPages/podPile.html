PodPile

About the Project

PodPile is a web app built with Next.JS that utilizes CRUD functionality. Users can store a list of their favourite podcasts, add new list items with details such as title, author, genre, and an image. Users can also edit and delete any podcast in their list. The app was designed to be responsive for web.

Technologies Used

Built with React and Next.JS, PodPile stores and manipulates it's data from data.json file via CRUD operations. The fetch calls are implemented when the user submits, deletes, or adds an item. The data are put into a global state by using the Context API which is responsible for the fetch calls. The app also uses standard React methods such as hooks.

Implementation

This App was built as a part of a school project and as such has not been publicly implemented. It was tested and graded on simulators and mobile devices.
