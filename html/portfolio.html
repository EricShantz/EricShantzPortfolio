Goobie

Bored with your friends and don't know what to do? Goobie provides a list of games that you and your friends can play together! Built with React Native, Goobie works on both Android and IOS and includes some fun Expo APIs to make the user experience all the better.

Music Player

This music player app is a single page application for Android built with Cordova. The app contains 2 scrollable lists, a search function/page, and a screen to display the current song playing along with all of its controls. The user can add any song to their "Top Picks" by clicking the heart icon on the song page.

Github User Search

This app connects to the GitHub Rest API and allows users to search for gitHub users. Users are able to filter results based on number of followers and can limit the amount of pages shown after a fetch. The app contains 5 individual pages and is designed for both landscape and portrait mode.

PodPile

PodPile is a web app built with Next.JS that utilizes CRUD functionality. Users can store a list of their favourite podcasts, add new list items with details such as title, author, genre, and an image. Users can also edit and delete any podcast in their list. The app was designed to be responsive for web.

Weather App

This weather app was a fun and simple project to work on using vanilla Javascript, HTML5, and CSS. The use of multiple RESTful APIs allows the user to see the weather based on their current location, or search for weather anywhere in the world. The app also displays general current information, as well as hourly and weekly weather forecasts. Each type of weather has its own dynamic background.
